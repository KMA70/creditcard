1. System Requirements

Operating System Required

  • Windows 10 / 11 (64-bit)
  • macOS 12 Monterey or later
  • Ubuntu 20.04 LTS or later

Hardware

  • RAM: 4 GB minimum (8 GB recommended)
  • Disk: 500 MB free space
  • CPU: Any modern x86-64 processor

Python Required

  • Python 3.8 or later (3.11 recommended)
  • pip (bundled with Python 3.8+)
  • Virtual environment tool: venv or conda

Browser Optional

  • Any modern browser for the Flask web UI
  • Google Chrome, Firefox, Edge, Safari

3. Installation Steps

Step 1 – Verify Python Installation

python --version
# Expected output: Python 3.x.x  (3.8 or later)

Step 2 – Create a Virtual Environment (Recommended)

# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate

Step 3 – Install Dependencies

cd credit_card_approval
pip install -r requirements.txt

Step 4 – Verify Dataset

Ensure the UCI Credit Card Approval dataset is present at data/crx.data.

# Check the file exists
ls data/crx.data        # macOS/Linux
dir data\crx.data       # Windows

If missing, download from the UCI ML Repository and place the file in the data/ folder.

Step 5 – Train the Model

python train_model.py

This generates models/best_model.pkl and models/model_meta.pkl required by the Flask app.

Step 6 – Run the Application

python app.py

Open your browser and navigate to http://localhost:5000.

6. No Cloud Account Required

Note: This project runs entirely on your local machine. No IBM Cloud, AWS, Azure, or any other cloud account is required. All ML inference is performed by the locally saved scikit-learn model.
